# Risp

- Inspired from [lispy](http://norvig.com/lispy.html) and
  [risp](https://stopa.io/post/222)

- **Status: Work in Progress**

- It has support for all basic operators, comparision operators as well as arithmetic operators.

## Keywords

- `define`
- `set!`
- `quote`

## Built-in functions

- `fmod`
- `mod`
- `and`
- `or`
- `not`
- `begin`
- `car`
- `cdr`
- `apply`
- `cons`
- `equal?`
- `length`
- `list?`
- `max`
- `min`
- `null?`
- `number?`
- `procedure?`
- `bool?`
- `map`
- `list`
